# GPT Image 2 Asset Prompt Pack

All prompts are for fictional, synthetic demo assets only. Do not use real children, real medical documents, real institutions, brands, phone numbers, addresses, IDs, or private credentials. Avoid clear identifiable child faces; use hands, backs, tabletop scenes, paper forms with `DEMO / 示例`, blurred activity context or illustrated scenes.

Common negative prompt for every asset:

`real identifiable child face, real person identity, real name, real phone number, real address, real medical record number, real hospital, brand logo, copyrighted character, gore, injury close-up, nudity, QR code, barcode, private credential, watermark from a real service`
