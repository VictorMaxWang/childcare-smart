# 产品规则缺口

本文件由 C99 汇总 `needs-product-spec` finding。C10-C23 发现产品规则不明确时，应先记录，不自行定义业务规则。

## 需要重点确认的规则

| 功能 | 待确认问题 |
| --- | --- |
| 家园沟通 | 家长回复是否直接发给老师；是否需要已读/待处理/已处理状态；是否支持附件。 |
| 成长档案 | 家长能否编辑；教师提交是否需要审核；记录按孩子还是班级归档。 |
| 成长绘本 | 生成结果保存多久；是否可重新生成；图片/音频版权和审核规则。 |
| 健康材料解析 | 上传文件类型、大小、隐私合规、解析失败处理、人工确认流程。 |
| 高风险会诊 | 风险等级、会诊角色、转办和关闭条件、家长可见范围。 |
| 营养餐谱 | 菜谱来源、AI 评价是否能覆盖营养师建议、历史记录周期。 |
| 周报 | 周报统计周期、发送对象、是否需要审批、是否可导出。 |
| 权限 | 园长、教师、家长对孩子、班级、机构级数据的边界。 |
